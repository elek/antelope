Chapter 13. Support

Please visit the project web pages at http://antelope.sourceforge.net for reporting bugs, user mailing lists, and other technical resources. This is also the location for finding the latest releases. In particular, if you are using Antelope as a jEdit plugin, you will invariably find newer versions on the Antelope project web site than from jEdit's plugin manager.

You can also contact me directly at <danson@germane-software.com>
