Antelope Roadmap

Roadmap

This page lists the proposed enhancements to Antelope and a rough timeline for implementation. Nothing is carved in stone, particular items may move up a version or be pushed back to a later date. This is simply a guide to what has been suggested and the order it will be included into Antelope.
Version Enhancement Date
3.5
  • Drag-and-drop build file builder.
 Jul 2004
3.4
  • View dependency graph using GEF.
 May 2004
3.3
  • Something similar to javadoc for build files. Probably integrate AntDoc.
 May/Jun 2004
3.2
  • Icon update.
  • Edit mode popup menu enhancement, open property file, show dependency tree, execute target.
  • Improve usefulness of "Resources" tab. Most items do not have useful values.
 May 2004
3.1
  • Icon update.
  • Assign shortcut key in jEdit to default target.
  • Display default target in different color than public or private targets.
  • Dependency tree in edit mode to show imported target dependencies.
 Apr 2004
3.0
  • Completely compliant with Ant 1.6.
  • Improved viewing of properties by adding a popup window to view long property values.
  • Show private targets as private in all modes.
  • View dependency tree in edit mode. Will eventually be replaced with GEF.
  • Icon update. Option to show icons, text, or both on control panel buttons.
  • Auto-reload of build file if a property file changes.
  • Allow opening of a property file for editing from the edit tree.
  • Add "reload" button to control panel.
 26 Jan 2004

Requested enhancements, but not yet planned:

  • Ability to print both build files and build output.
  • Integration with Eclipse.
